### ðŸ”¹ **1. Baholash tizimi**

### ðŸ”¹ **2. Sodda kalkulyator**

- Noto'g'ri amal kiritilsa: `"Noto'g'ri amal. Faqat +, -, *, / ishlatiladi."`

**Misol:**
```
1-son: 10
2-son: 3  
Amal: +
Natija: 10 + 3 = 13
```
